About me

This is a website created with HTML5 and CSS3 using JavaScript ES6. You can browse the whole git history of this web project on gitlab repository. To view it, ask the owner of this website for link.

You can use interactive javascript implementations, such as "Reverse the input", "Sum and multiplication of number", "Choose page background color".

Also you can hover the logo in the header to see the modal screen with the information about author.

Basically, this website was created under the 4th semester of Computer Science during Basics of Internet Technologies course which are led by Готинчан Тетяна Іванівна in 2022, quarantine studying mode.
